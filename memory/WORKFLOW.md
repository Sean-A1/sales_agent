# WORKFLOW.md — 개발 워크플로

> 팀 개발 프로세스 및 작업 흐름을 기록합니다.

## Git Workflow

1. `main`에서 최신 코드 pull
2. `sean/<task>` 또는 `claude/<task>` 브랜치 생성
3. 작업 → 커밋 → PR 생성
4. 리뷰 후 `main`에 머지
